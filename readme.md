Este proyecto es una página que le permite al usuario visualizar una variedad de productos (detalles) guardados en una base de datos mongo. Con esta api podemos crear, editar, eliminar, obtener un producto y obtener todos los productos de la base de datos.

Juan Diego Pardo Camargo es el único autor de este proyecto.

Para poder ejecutar este proyecto necesité:
*MongoDB/mongoose para manejar la base de datos.
*express para hacer la api.
*cors para accesibilidad.
*dotenv para usar el archivo con las variables de entorno.
*nodemon y node.js para el desarrollo


Para poder levantar el backend debemos seguir estos pasos:
    *Entrar a la carpeta del backend (cd backend).
    *Instalar dependencias (npm install).
    *Correr el servidor (npm start).

Si todo está bien, en la consola deberías ver algo similar a:

Conectado a MongoDB Atlas
Servidor escuchando en http://localhost:3000
